# Программа персонального чата  

## Описание    
Пользователи могут обмениваться сообщениями в режиме реального времени.  
Этот проект состоит из двух программ:  

1. `first_user`   
2. `second_user`  

## Как использовать    

1. **Компиляция программы:**    
- make    

2. **Запуск программы:**    
- ./first_user  
откройте второй терминал и запустите:  
- ./second_user
