# Простой файловый менеджер

Это простой файловый менеджер, который позволяет просматривать содержимое каталогов и отображает типы файлов.

1. **Компиляция программы:**
- make

2. **Запуск программы:**
- ./program 

3. **Примеры использования:**
- ./program
- `'` для текущей директории
- например /home
